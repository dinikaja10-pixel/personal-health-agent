# Privacy Policy – Personal Health Agent

Last updated: July 21, 2026

Personal Health Agent is a personal development project designed to access and analyze health and fitness data for personal use.

## Data Access

The application may access health and fitness data provided through connected services such as WHOOP, including recovery, sleep, workout, cycle, profile, and body measurement data.

## Use of Data

The accessed data is used solely to provide personal health insights, analysis, and recommendations.

Personal data is not sold, rented, or shared with third parties for advertising or marketing purposes.

## Data Security

Authentication credentials and API tokens are handled securely and are not intentionally made publicly available.

## Contact

For questions regarding this Privacy Policy, please contact the application owner through this GitHub repository.
